# Deck

AI-powered pitch deck generator. Create beautiful 7-slide pitch decks in seconds.

## Features

- Clean, professional design (Fraunces + Inter fonts)
- AI-generated content with real statistics
- Keyboard navigation (arrow keys, F for fullscreen)
- Export to PNG and PPTX (coming soon)

## Getting Started

1. Clone the repo
2. Install dependencies:
   ```bash
   bun install
   ```
3. Add your Anthropic API key to `.env.local`:
   ```
   ANTHROPIC_API_KEY=your_key_here
   ```
4. Run the dev server:
   ```bash
   bun dev
   ```
5. Open http://localhost:3000

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Claude AI (Anthropic)
